# Portal Backend for Yuta management

[![pdm-managed](https://img.shields.io/badge/pdm-managed-blueviolet)](https://pdm.fming.dev)

<div>

## Prerequisites

- Python 3.10 or above.
- `pip`, `pdm` installed.

## Quickstart

Install the required packages:

```
pdm install
```

Running with dev mode:

```
pdm run dev
```
